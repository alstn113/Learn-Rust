### rustup으로 컴파일러와 패키지 매니저 설치

    https://rustup.rs/

### Rust 버전 확인하기

    rustc --version

### rustup 업데이트

    rustup --update

### VSC Extension

    rust-analyzer, Better TOML, crates, CodeLLDB 설치

### 새 프로젝트 시작하기

    cargo new <프로젝트 이름>

### 러스트 빌드, 실행

    cargo build
    cargo run
